# Directorio de Imágenes - Portal Ñapanchita

## Estructura

```
images/
├── logo.png            ✓ Logo principal del restaurante
├── carousel-1.jpg      - Imagen 1 del carrusel (usa fallback de Unsplash)
├── carousel-2.jpg      - Imagen 2 del carrusel (usa fallback de Unsplash)
├── carousel-3.jpg      - Imagen 3 del carrusel (usa fallback de Unsplash)
├── staff.jpg           - Foto del equipo (usa fallback de Unsplash)
├── platos/             ✓ Imágenes de platos del menú
└── combos/             ✓ Imágenes de combos
```

## Imágenes del Carrusel

El carrusel en la página principal muestra 3 imágenes con mensajes clave:

1. **carousel-1.jpg**: Ceviches frescos - "Los Mejores Ceviches de la Ciudad"
2. **carousel-2.jpg**: Mariscos del día - "Mariscos Frescos del Día"
3. **carousel-3.jpg**: Delivery rápido - "Delivery Express"

## Imagen del Staff

**staff.jpg**: Foto del equipo de Ñapanchita para la sección "Sobre Nosotros"

## Fallbacks

Si las imágenes no existen, el sistema utiliza automáticamente imágenes de alta calidad de Unsplash relacionadas con:
- Ceviches y comida peruana
- Mariscos frescos
- Delivery de comida
- Equipos de restaurante

## Recomendaciones

Para mejores resultados, reemplaza las imágenes con fotos reales del restaurante:
- **Resolución**: Mínimo 1200x600px para carrusel, 800x800px para staff
- **Formato**: JPG optimizado para web
- **Peso**: Máximo 500KB por imagen
- **Aspecto**: 16:9 para carrusel, 1:1 o 4:3 para staff
